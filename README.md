
# QA Engineering Learning Journey

## Day 1 Project
### Data Quality Checker using Python

This project simulates a QA/data validation workflow using Python.

The script checks:
- Missing values
- Invalid score ranges
- Negative ages
- Invalid email formats
- Incorrect status formatting

It also performs:
- Statistical analysis
- Status distribution analysis
- Basic reporting
